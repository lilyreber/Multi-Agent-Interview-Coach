# Отчет 
Была реализована система, состоящая из нескольких  AI-агентов
* Агент-Менеджер принимает решени о найме
* Агент-Интервьюер должен проводить интервью, но не проверять факты
* Агент-Критик, должен проверять факты и давать рекомндации

В итоге получили рабочего собеседующего агента, который даёт полный фидбек в необходимом формате, может отличать бредовые ответ от правильных, задаёт вопросы соотсветсвующие вакансии и опыту интервьюируемого, также умеет подстраиваться под его уровень знаний и сложности
